# picoPlaca
Pico y Placa demostration source code

## Install
```
mvn clean install
```

## Run
```
java -jar .\target\picoplaca-1.0-SNAPSHOT-jar-with-dependencies.jar
```

## Console output

Input date and hour: (yyyy-MM-dd)  example: 2020-01-10 14:30
